== Description ==

The TrustCard Wordpress Widget is the simplest way to integrate your TrustCard into your site or blog. Like a portable online ID card for the Sharing Economy, the TrustCard is a snapshot into a your TrustCloud profile.

== Installation ==

1. Upload `tc-trustcard.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. From Appearance > Widgets, drag the TrustCard widget into one of your widget areas
4. Expand the widget item, enter your Twitter handle or email address, and save
